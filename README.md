# Secure Login System

## Objective
Develop a secure login web application with password hashing and session management.

## Features
- User Registration
- User Login
- Password Hashing
- Session Management
- Logout Feature

## Technology Used
- Python
- Flask
- Werkzeug

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run:
   python app.py

3. Open:
   http://127.0.0.1:5000
